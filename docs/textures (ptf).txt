P3M - PlatinumSrc Texture File/Format

- .ptf file extension
- Current version is 0.0
- All data should be little endian
- LZ4F should be used for compression

Format:

    <Header> <Data>

    Header:
        <char[3]: {'P', 'T', 'F'}>
        <u8: Major version>
        <u8: <3 bits: 0> <1 bit: Has alpha> <4 bits: Resolution (2^n)>>
    Data:
        { <Compressed <u8[3]: RGB>...>
        | <Compressed <u8[4]: RGBA>...> }
